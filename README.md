# README

## Modification de la Base de Données (BNGRC/OCHA)

Bienvenue dans cette version modifiée de la Base de Données initialement développée par BNGRC/OCHA.
Cette adaptation a été réalisée dans le but de répondre à des besoins spécifiques et d'améliorer les fonctionnalités existantes.

### Avertissement

Cette base de données repose sur le travail initial de BNGRC/OCHA en 2018, et nous reconnaissons et respectons leurs contributions à la création de la version originale. 
Tous les crédits pour la conception et la mise en place de la base de données initiale reviennent à l'équipe de BNGRC/OCHA.

### Modifications Apportées

Les modifications apportées à cette base de données ont été effectuées dans le but d'ajuster ses fonctionnalités pour répondre aux exigences particulières de notre projet.
Les principales modifications incluent, mais ne se limitent pas à :

  - Ajout de fonctionnalités spécifiques.
  - Optimisation de la structure de la base de données.
  - Mise à jour des régions de Madagascar, passant de 22 à 23 régions.

### Comment Utiliser Cette Version

Pour utiliser cette version modifiée de la Base de Données initiale, suivez les étapes ci-dessous :

1. **Extension Utilisée :** Le format Geopackage est utilisé.
2. **Contenu du Dossier :** Ce dossier contient 4 couches Geopackage :
    - `madagascar-mdg-adm0-country-***.gpkg` : couche de Madagascar uniquement.
    - `madagascar-mdg-adm1-region-***.gpkg` : délimitation des 23 régions de Madagascar.
    - `madagascar-mdg-adm2-district-***.gpkg` : délimitation administrative des districts de chaque région de Madagascar.
    - `madagascar-mdg-adm3-communes-***.gpkg` : délimitation administrative des communes de chaque district.
    - `madagascar-mdg-adm4-fokontany-***/.gpkg` : délimitation administrative des fokontany dans chaque commune.
